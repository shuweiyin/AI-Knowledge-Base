# 🧰 Skills Index — Reusable Teaching Templates

> This folder contains **6 reusable skill templates** that form a complete teaching suite for any AI topic.  
> Each template is self-contained *and* cross-links to the others — creating a fully connected topic web.

---

## 📐 How the Naming System Works

**Every template uses `{{TOPIC}}` as a placeholder.**  
When you create real notes, follow this exact naming pattern:

```
{{TOPIC}} - Module Overview
{{TOPIC}} - Concept Note
{{TOPIC}} - Lesson Plan
{{TOPIC}} - Discussion
{{TOPIC}} - Project
{{TOPIC}} - Quiz
```

**Example** — for the topic *"Machine Learning"*:

| File Name | Purpose |
|-----------|---------|
| `Machine Learning - Module Overview` | Curriculum blueprint for the whole topic |
| `Machine Learning - Concept Note` | Deep-dive explanation of the core idea |
| `Machine Learning - Lesson Plan` | Step-by-step classroom lesson |
| `Machine Learning - Discussion` | Socratic discussion questions |
| `Machine Learning - Project` | Hands-on student project |
| `Machine Learning - Quiz` | Formative assessment & rubric |

Because all 6 files share the same `{{TOPIC}}` prefix, every wikilink **auto-resolves** as soon as you create the files.

## 🚀 Quick Start: Using a Template

1. **Pick a topic** (e.g., *"Neural Networks"*)
2. **Open** any skill template file below
3. **Copy** the entire note content
4. **Create a new note** named exactly: `Neural Networks - Lesson Plan` (or whichever type)
5. **Paste** the content and replace every `{{TOPIC}}` with `Neural Networks`
6. **Fill in** the `[bracketed placeholder]` sections with your actual content
7. **Repeat** for the other 5 templates — all wikilinks will resolve automatically
8. **Update the Glossary** — every new term in the "Key Terms" section must have an entry in [[Glossary]] ← mandatory

## 📖 Glossary Rule

> **Every term introduced in a skill template or generated note must be added to the Glossary.**

| When | What to do |
|------|-----------|
| You add a term to "Key Terms" in a Concept Note | Add it to [[Glossary]] with a one-sentence definition |
| You introduce a new concept in a Lesson Plan | Add the concept name + definition to [[Glossary]] |
| A student asks "what does X mean?" | If X isn't in [[Glossary]], add it |
| Claude generates content with jargon | Check every technical word against [[Glossary]] |

**Format for Glossary entries:**
```
**Term** — Definition at middle-school reading level. → [[Source Note]]
```
