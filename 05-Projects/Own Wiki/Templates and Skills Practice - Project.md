## ЁЯУж Step-by-Step Instructions

### Step 1 тАФ Choose Your Topic & Plan the Suite *(~15 minutes)*

**What to do:**
Pick one AI topic you want to learn deeply. This topic will become your full 6-file suite.

**Suggested topics** (or choose your own):
- `Machine Learning`
- `Neural Networks`
- `Natural Language Processing`
- `Computer Vision`
- `AI Ethics`
- `Prompt Engineering`
- `Supervised Learning`
- `Recommendation Systems` *(great for connecting to Netflix, Spotify, YouTube)*

**Steps:**
1. Browse [[00-SKILLS-INDEX]] тАФ read the naming convention section carefully
2. Choose your topic. Write it down exactly as you'll use it (capitalization matters!)
3. Create a planning note named `[Your Topic] - Planning` with this content:
   ```
   Topic: [Your Topic]
   
   My 6 files will be:
   - [Your Topic] - Module Overview
   - [Your Topic] - Concept Note
   - [Your Topic] - Lesson Plan
   - [Your Topic] - Discussion
   - [Your Topic] - Project
   - [Your Topic] - Quiz
   
   What I already know about this topic:
   [2-3 sentences]
   
   What I want to learn:
   [2-3 questions]
   ```
4. Ask Claude: *"I'm going to build a 6-note topic suite about [Your Topic] for high school students. What are the 5 most important concepts I should cover?"*

**Expected result:** A clear topic choice and planning note. You know the 5 key concepts before writing.

---

### Step 2 тАФ Create the 6 Files with Correct Names *(~20 minutes)*

**What to do:**
Create all 6 notes with the exact naming pattern. Empty is fine тАФ you'll fill them in the next steps.

1. Open [[00-SKILLS-INDEX]] and keep it visible in a split pane
2. Create 6 new notes in your most relevant topic folder (e.g., `01-AI-Fundamentals/`)
3. Name them **exactly**:
   - `[Your Topic] - Module Overview`
   - `[Your Topic] - Concept Note`
   - `[Your Topic] - Lesson Plan`
   - `[Your Topic] - Discussion`
   - `[Your Topic] - Project`
   - `[Your Topic] - Quiz`
4. Copy the content from each skill template (Templates 1тАУ6) into each file
5. Replace every single `{{TOPIC}}` with your actual topic name

**Expected result:** 6 notes exist, all named correctly, all containing the template structure with `{{TOPIC}}` replaced.

**Checkpoint:** Open Graph View. Do you see 6 dots labeled with your topic? Are they connected to each other? (They should be тАФ the cross-link table at the bottom of every template does this automatically!)

**Common mistake:** Typo in topic name (e.g., `Machine learning` vs `Machine Learning`) breaks wikilinks тЖТ **Fix:** Search-and-replace consistently. Wikilinks are case-sensitive in Obsidian.

---

### Step 3 тАФ Fill in the Concept Note First *(~40 minutes)*

**What to do:**
Always start with the **Concept Note** тАФ it is the foundation all other files reference.

1. Open `[Your Topic] - Concept Note`
2. Work through each section of the template, filling in real content:
   - Write the one-sentence core concept in your own words first
   - Ask Claude: *"Give me 2 strong analogies for [Your Topic] that a 15-year-old would immediately understand"*
   - Ask Claude: *"What are the 3 most common misconceptions students have about [Your Topic]?"*
   - Ask Claude: *"Define these 5 key terms for [Your Topic] in one sentence each: [list terms]"*
3. **Critical rule:** Always read Claude's output, edit it, and make sure *you* understand it before keeping it
4. Fill in the "Try It" activity тАФ design something hands-on and specific to your topic
5. Fill in the "Ethics Connection" тАФ every AI topic has one

**Expected result:** A complete Concept Note that *you* can explain to a classmate without reading from the note.

**Self-check question:** *Can you close the note and explain the concept in 60 seconds to someone next to you? If not тАФ the note isn't done yet.*

---

### Step 4 тАФ Fill in the Lesson Plan & Discussion *(~35 minutes)*

**What to do:**
Build the classroom experience for your topic.

**Lesson Plan (`[Your Topic] - Lesson Plan`):**
1. Write the Hook тАФ make it something that would genuinely surprise or provoke a student
2. For "Direct Instruction," write bullet-point teacher notes (what you would say, not just what the concept is)
3. Design the Guided Practice activity тАФ must involve students *doing* something, not just listening
4. Write the Exit Ticket questions тАФ these should match your learning objectives
5. Ask Claude: *"Review this lesson plan hook for [Your Topic]. Is it engaging enough for a 10th grader?"*

**Discussion Guide (`[Your Topic] - Discussion`):**
1. Write at least one question at each of the 3 levels (Recall, Analysis, Ethics)
2. For the Level 3 ethics question тАФ think about a real controversy or dilemma connected to your topic
3. Write "Possible student positions" for the ethics question тАФ think of at least 3 angles
4. Ask Claude: *"Give me a provocative Level 3 discussion question about [Your Topic] that teenagers would genuinely argue about"*

**Expected result:** A lesson plan you could actually teach from, and a discussion guide with 5+ questions across 3 levels.

---

### Step 5 тАФ Complete the Module Overview, Project & Quiz *(~40 minutes)*

**What to do:**
Finish the full suite.

**Module Overview (`[Your Topic] - Module Overview`):**
- Write the "Big Question" тАФ one sentence that captures why this topic matters
- Fill in the weekly structure тАФ 3 weeks, what would you cover each week?
- Fill in the assessment plan table using the correct links to your other 5 files

**Project (`[Your Topic] - Project`):**
- Design a real hands-on project for your topic
- Make sure it's achievable in 3тАУ4 class periods with free tools
- Write the Step-by-Step with honest "common mistakes" for each step
- Fill in the rubric тАФ make the criteria specific to your topic

**Quiz (`[Your Topic] - Quiz`):**
- Write 4 multiple-choice questions (1 on each difficulty level: recall тЖТ ethics)
- Write 2 short-answer questions with model answers
- Write 1 critical thinking question with a full rubric
- Ask Claude to check: *"Are these quiz questions testing different levels of thinking, or are they all the same level?"*

**Expected result:** All 6 files are complete. The graph shows a tightly connected 6-node web for your topic.
