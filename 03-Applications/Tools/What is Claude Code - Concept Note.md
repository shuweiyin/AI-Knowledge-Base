# What is Claude Code — Concept Note

## ⚡ Core Concept (One Sentence)

> Claude Code is an **AI assistant made by Anthropic** that lives inside your computer's workspace — it can read your files, write code, edit notes, and help you build projects by understanding your *entire* project at once.

---

## 🤔 Why It Matters

Most AI chatbots (like a basic ChatGPT window) only know what you type into the chat box. Claude Code is different — it can *see your whole project*: every file, every note, every folder. This means it can give much smarter, more specific help. For students building an AI knowledge base, Claude Code is like having an expert tutor who has already read all your notes before you even ask a question.

---

## 📖 Detailed Explanation

### Part 1 — The Basics

Claude Code is a **coding and knowledge assistant** built by Anthropic (the AI safety company). Unlike a website chatbot, Claude Code runs directly in your **terminal** (a text-based interface on your computer) or through an editor plugin like the **Claudian plugin** inside Obsidian.

The key idea: Claude Code has access to your **file system** — with your permission, it can read, create, and edit files on your computer. This makes it far more powerful than a chatbot for real project work.

### Part 2 — How It Works (Step by Step)

1. **You start Claude Code** in your project folder (vault, codebase, etc.)
2. **It reads the context** — including special files like `Claude.md` that tell it about your project and goals
3. **You give it a task** — in plain English: *"Write a lesson plan for machine learning"* or *"Fix the broken links in this note"*
4. **It uses tools** to act: reading files, writing files, searching the web, running code
5. **It shows you what it plans to do** before doing it — you stay in control
6. **It executes** and you review the result

### Part 3 — Real-World Applications

- **Students** use it to generate lesson notes, study guides, and flashcards from their existing notes
- **Teachers** use it to build curriculum materials, quizzes, and project briefs at speed
- **Developers** use it to write, debug, and refactor code across entire codebases
- **Researchers** use it to summarize papers, find connections, and draft reports
- **This vault** was built *with* Claude Code — the templates, structure, and notes were all co-created with it

## 🪄 Analogies & Stories

**Analogy 1 — The Smart Colleague 🧑‍💼**
Imagine hiring a brilliant colleague who, on their first day, read *every document in the office* overnight. When you ask them a question, they already know the context. They don't ask "which file?" — they already know. That's Claude Code: it reads your whole vault before you even start.

**Analogy 2 — The GPS vs. the Map 🗺️**
A regular AI chatbot is like a paper map — useful, but you have to describe where you are. Claude Code is like a GPS — it *knows* exactly where you are (your files, your project) and gives directions that fit your exact situation.

## ❓ Common Student Questions

**Q: Is Claude Code the same as ChatGPT?**
A: Both are AI assistants, but they're different products from different companies (Claude Code is from Anthropic, ChatGPT is from OpenAI). The bigger difference is *how* you use them: ChatGPT is mainly a web chat. Claude Code is a workspace tool that integrates into your files and projects.

**Q: Does Claude Code send my files to the internet?**
A: Yes — when Claude Code reads your files to answer a question, that content is sent to Anthropic's servers to generate a response. This is why the `Claude.md` file matters: you control *what context* Claude sees. Never put passwords or deeply private information in notes you use with Claude Code.

**Q: Do I need to know how to code to use Claude Code?**
A: No! You can use it in plain English. It can write code *for* you, explain code to you, or help you with non-code tasks like organizing notes, writing lesson plans, or summarizing ideas.

**Q: What's the difference between Claude Code and the Claudian plugin?**
A: Claude Code is the full standalone tool (runs in terminal). The [[Claudian plugin]] is a lighter version built into Obsidian — it lets you use Claude directly inside your vault without opening a terminal. Both use the same Claude AI underneath.

## ⚠️ Common Misconceptions

| Misconception | The Truth |
|--------------|-----------|
| "Claude Code writes code for you so you don't need to learn" | It's a *collaborator*, not a replacement — you still need to understand and review what it produces |
| "It knows everything about everything" | It has a knowledge cutoff date and can make mistakes — always verify important facts |
| "It's always connected to the internet" | By default it works from its training knowledge; web search is a tool it can use when enabled |
| "Using AI to help write notes is cheating" | AI is a tool like a calculator — knowing how to use it well is itself a skill worth learning |

## 🔑 Key Terms & Definitions

- **[[Large Language Model (LLM)]]**: The AI technology behind Claude — trained on massive amounts of text to understand and generate language
- **[[Context Window]]**: How much text Claude can "see" at one time — like short-term memory. Larger = smarter responses about your project
- **[[Claude.md]]**: A special file in your vault that tells Claude about your project, goals, and how you want it to behave — the most important file in this vault
- **[[System Prompt]]**: Instructions given to an AI before a conversation starts, shaping how it responds (Claude.md functions like a system prompt)
- **[[MCP (Model Context Protocol)]]**: A standard way for Claude to connect to external tools — file systems, calendars, web search, and more
- **[[Anthropic]]**: The AI safety company that built Claude — their mission is to make AI that is safe, honest, and beneficial

## 🖐️ Try It — Quick Activity

**Activity:** Ask Claude Something About Your Own Notes
**Time:** 5–10 minutes
**Materials:** Obsidian + [[Claudian plugin]] installed

**Instructions:**
1. Open any note in your Obsidian vault
2. Open the Claudian plugin panel
3. Ask: *"Can you summarize this note in 3 bullet points?"*
4. Then ask: *"What topic would be a good next note to write, based on what I have so far?"*
5. Try one more: *"What wikilinks am I missing in this note?"*

**Reflect:** *"What surprised you about how Claude responded? Did it seem to understand your notes, or did something feel off?"*

## 🤝 Ethics Connection

Claude Code raises important questions about **authorship, privacy, and dependency**. If Claude helps write your notes, who is the author — you or the AI? What responsibility do you have to verify the content it generates? And what happens if you become so reliant on AI that you struggle to think through problems alone?

These aren't reasons to avoid Claude Code — they're reasons to use it *thoughtfully*. The goal is for AI to amplify your thinking, not replace it. Always ask: *"Do I actually understand what Claude just wrote?"*
