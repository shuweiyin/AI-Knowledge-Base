# 🟢 基础班 — 在线课程介绍

> *面向11岁及以上学生的五周入门课程，无需任何编程基础。*

---

## 📢 致家长 — 为什么这门课程很重要

我们生活的世界正在被人工智能（AI）深刻改变。AI 已经在决定您的孩子看哪些视频、哪些求职申请会被优先审阅、医生如何更早发现疾病。理解 AI，已不再是专业人士的专属技能——它正在成为像阅读和写作一样基础的能力。

**这门课程的目标不是让您的孩子一夜之间成为程序员。** 我们给予他们更有价值的东西：自信地使用 AI 工具的能力、用批判性思维理解和讨论 AI 的语言，以及亲手创造作品的成就感。

在这五节课中，您的孩子将：

- ✅ 写出人生中第一行代码——通过音乐、舞蹈的方式（从积木式编程开始，无需担心打字）
- ✅ 理解 AI 如何识别规律、从数据中学习并做出决策
- ✅ 使用全球领先的 AI 助手 Claude，从零开始亲手制作一款真正可以玩的游戏
- ✅ 通过亲眼看到程序中的循环、数组和碰撞检测在自己制作的游戏中运作，掌握核心计算机科学概念
- ✅ 培养"提示词工程"技能：学会向 AI 给出清晰、具体的指令，获得有用的结果

**这与您孩子的未来有什么关系？**
无论您的孩子将来成为工程师、医生、艺术家还是创业者——他们都将与 AI 并肩工作。那些理解 AI 运作原理（而不仅仅是会用）的学生，将拥有持久的竞争优势。这门课程正是为此打下基础。

**孩子学完后将收获什么？**
- 一款由他们亲手用 AI 制作的 **AI 知识问答游戏**，完全可以玩
- 一款加入了自己选择功能的 **贪吃蛇游戏**
- 一张 **Code.org 结业证书**
- 能够用自己的话描述 AI 是什么、如何学习、为何重要

## 📅 逐节课程安排

### 第一节 — 欢迎来到编程与 AI 的世界 *（1小时）*

> **本节核心问题：** *编程究竟是什么感觉——它和音乐又有什么关系？*

|          |                                                                                     |
| -------- | ----------------------------------------------------------------------------------- |
| **本节内容** | 简短友好的课程介绍，讲解 AI 到底是什么。随后立刻进入实践：学生使用循环（loops）创作音乐，用事件（events）为动画舞者编写动作程序。无需打字——只需思考。 |
| **核心概念** | 什么是 AI · 循环（Loops）· 事件（Events）· 序列（Sequences）                                       |
| **学生收获** | "我写出了第一个程序，还用它创作了一件有趣的作品。"                                                          |

---

### 第二节 — 跟着节拍动起来 *（1小时）*

> **本节核心问题：** *如何告诉电脑按正确的顺序做事——如果顺序错了会发生什么？*

|          |                                                                                                                                                              |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **本节内容** | 学生将第一节学到的音乐和舞蹈结合起来——这次为一个 AI 机器人编写动作和表演程序。重点在于**排序（Sequencing）**：将指令按正确顺序排列，让机器人精确完成表演。学生将亲自发现，当步骤顺序错误时会发生什么，并理解"顺序"为何是计算机科学中最基础的概念之一。随后，探索 AI 如何在海量数据中寻找规律。 |
| **核心概念** | 排序（Sequencing）· 机器学习（Machine Learning）· 大数据（Big Data）· 模式识别                                                                                                  |
| **学生收获** | "指令的顺序非常重要——这个道理适用于所有程序。"                                                                                                                                    |

---

### 第三节 — 认识 Claude · 制作知识问答游戏 *（1小时）*

> **本节核心问题：** *我能不能用普通话描述一个游戏，然后让 AI 在一分钟内把它做出来？*

| | |
|---|---|
| **本节内容** | 介绍 Claude 和 Artifacts 是什么。学生用一段结构化的"提示词"，直接在浏览器中生成一款完整可玩的 AI 知识问答游戏——带有动画效果、得分追踪和即时反馈。随后，学生将游戏个性化，替换成自己写的问题。 |
| **核心概念** | 什么是 Claude · 提示词工程（Prompt Engineering）· Artifacts · HTML · CSS · JavaScript（初步认识，无需掌握） |
| **使用平台** | [Claude.ai](https://claude.ai) |
| **学生收获** | "我只是描述了我想要的游戏，AI 就帮我做了出来，还是真正属于我的。" |

---

### 第四节 — 贪吃蛇游戏 · 制作它 *（1小时）*

> **本节核心问题：** *贪吃蛇是怎么运作的——它和 GPS 导航、自动驾驶汽车又有什么共同点？*

| | |
|---|---|
| **本节内容** | 动手制作之前，学生先了解贪吃蛇背后的计算机科学原理：数组（Array）如何追踪蛇身的每一节、游戏循环（Game Loop）如何每秒运行十多次、碰撞检测（Collision Detection）如何判断游戏结束。然后，用 Claude Artifacts 生成一款功能完整的贪吃蛇游戏，并按照检查清单逐项测试。 |
| **核心概念** | 游戏循环（Game Loop）· 数组（Arrays）· 碰撞检测（Collision Detection）· Canvas API |
| **使用平台** | [Claude.ai](https://claude.ai) |
| **学生收获** | "我不只会玩这个游戏——我还理解了它内部是怎么运作的。" |

---

### 第五节 — 贪吃蛇游戏 · 升级它 + 展望未来 *（1小时）*

> **本节核心问题：** *如何让这款游戏变成专属于我的——接下来我还能做什么？*

| | |
|---|---|
| **本节内容** | 学生从功能菜单中选择两项改进（例如：穿墙传送模式、粒子爆炸特效、排行榜、AI 自动玩）并用提示词实现它们。然后进行"代码寻宝"——在真实源代码中找到 `snake.push`、`setInterval`、`localStorage` 等关键概念。最后进行学习反思，并预告下一阶段：工作室班。 |
| **核心概念** | 迭代改进 · 阅读代码 · 路径寻找算法（Pathfinding，初步了解）· 本地存储（localStorage） |
| **使用平台** | [Claude.ai](https://claude.ai) |
| **学生收获** | "我改进了自己的游戏，而且我真的能看懂一部分代码了。" |

- 对 AI 和大数据概念的了解
